# Strings
Strings are treated as arrays in C.

'\0' Null character is necessary for marking the end of the string. (not Null Pointer)

``` C
//Declaring a string

char c[] = "MINERIC";  // {'M', 'I', 'N', 'E', 'R', 'I', 'C', '\0'};
char c[] = {'M', 'I', 'N', 'E', 'R', 'I', 'C', '\0'};  // "MINERIC"
```
